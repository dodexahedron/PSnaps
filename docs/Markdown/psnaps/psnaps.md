# psnaps Module

## Description

A PowerShell module and .net library for management of Snap packages on Linux. Requires PS 7.5.4 or greater and .net 9.0 or greater.

## psnaps Cmdlets

### [Get-SnapdClient](Get-SnapdClient.md)

Gets the current snapd client in use by the PSnaps module.

### [Get-SnapPackage](Get-SnapPackage.md)

Gets one or more snap packages, according to input options.

### [Install-SnapPackage](Install-SnapPackage.md)

Installs one or more specified snap packages.

### [Set-SnapdClient](Set-SnapdClient.md)

Sets the client object the PSnaps module will use.

### [Uninstall-SnapPackage](Uninstall-SnapPackage.md)

Uninstalls one or more installed snap packages from the system.
